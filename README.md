# Rolling Ball Game

## Description

Rolling Ball Game is a simple Unity game where the player controls a ball and moves around a platform to collect targets.

## Gameplay

* Use the keyboard arrow keys to move the ball.
* Move around the green platform and touch the cubes.
* When the ball touches a cube, the cube disappears.
* The goal is to collect all the cubes.

## Controls

* Up Arrow → Move Forward
* Down Arrow → Move Backward
* Left Arrow → Move Left
* Right Arrow → Move Right

## Game Over

* If the ball falls off the green platform, the game is over.

## Features

* Physics-based ball movement
* Collectible cubes
* Simple obstacle-free environment
* Game over when falling off the platform

## Built With

* Unity
* C#

## How to Run

1. Open the project in Unity.
2. Open the main scene.
3. Press Play.
